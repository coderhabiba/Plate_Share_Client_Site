## üåü Key Features

PlateShare is a full-stack food donation and distribution platform built with robust features to connect donors and recipients:

* **Easy Food Donation:** Donors can easily add new food items to the system (**Add Food**), including details like name, quantity, expiry date, and images.
* **Transparent Donation Management:** Donors can view all **Requests** received for their donated food items and have the power to **Accept** or **Reject** these requests as needed.
* **Secure User Authentication:** Users can register using email/password (**Register**) and utilize the convenient **Google Login** option for quick and seamless access.
* **Dynamic Food Status & Updates:** Donors can **Update** the information of their donated food at any time. Recipients can also track the status of the requests they have made.
* **Featured Food Section:** The website includes a dedicated **Featured Food** section that highlights items ready for quick distribution, ensuring timely sharing.
